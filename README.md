# OptimusAutomate_ImageClassification
# Image Classification with CNN

## Project Overview

This project implements a Convolutional Neural Network (CNN) using TensorFlow to classify handwritten digits from the MNIST dataset.

## Features

* CNN-based image classification
* MNIST dataset training
* Accuracy visualization
* Confusion matrix generation
* Test accuracy of approximately 98.9%

## Technologies Used

* Python
* TensorFlow
* NumPy
* Matplotlib
* Scikit-learn
* Seaborn

## Results

The model achieved approximately 98.9% test accuracy on the MNIST test dataset.
